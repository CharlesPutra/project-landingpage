Rekayasa Perangkat Lunak (RPL)

RPL adalah jurusan yang mempelajari tentang cara membangun perangkat lunak atau aplikasi berbasis desktop, web, dan mobile. Jurusan ini sangat cocok untuk kamu yang suka ngoding, teknologi, dan pengembangan sistem.

Di jurusan ini, siswa akan belajar pemrograman, desain UI/UX, basis data, manajemen proyek software, serta pemahaman tentang teknologi modern seperti AI, IoT, dan Cloud Computing. Lulusan RPL bisa bekerja sebagai programmer, web developer, software engineer, dan profesi teknologi lainnya.

Apa yang Dipelajari di RPL?

Bahasa Pemrograman

Belajar berbagai bahasa seperti HTML, CSS, JavaScript, PHP, Python, hingga Java dan C++ untuk membangun aplikasi.

Pengembangan Aplikasi

Membuat aplikasi desktop, website, dan mobile dari awal dengan menerapkan logika dan struktur data yang tepat.

Database & UI/UX

Belajar membuat dan mengelola database (MySQL, MongoDB) serta membuat tampilan antarmuka yang menarik dan user-friendly.
